# Student Marks Analysis

This project analyzes student marks data using Python.

## Tools Used
- Python
- Pandas
- Matplotlib

## Project Work
- Loaded student dataset
- Cleaned missing or incorrect data
- Calculated average marks of students
- Compared performance of students
- Created visualizations like bar charts

## Key Insights
- Identified top performing students
- Found average class performance
- Compared subject-wise scores

## Conclusion
This project helps in understanding student performance using data analysis techniques.
